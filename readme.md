## Commands

| Command              | Description                   |
| -------------------- | ----------------------------- |
| `npm run start`      | start dev server              |
| `npm run watch`      | watch for file changes        |
| `npm run build`      | build for production          |
| `npm run fonts`      | convert fonts                 |
| `npm run fontsStyle` | include fonts in css          |
| `npm run images`     | compress images               |
| `npm run js`         | process and minify js files   |
| `npm run css`        | process and minify scss files |
| `npm run html`       | process and build html files  |
| `npm run svg`        | process svg files             |
